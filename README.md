# Archi-Sketch

Archi-Sketch is a web-based tool designed to support early-stage architectural design by providing instant cost estimation based on key building parameters.

## Overview

The system allows users to input basic design details such as building type, size, location, and specification level. Based on these inputs, the application generates an approximate cost estimate along with a breakdown of key cost components.

This tool is intended to help users understand the financial impact of design decisions at an early stage.

## Features

- User-friendly interface for entering building parameters  
- Rule-based cost estimation system  
- Cost breakdown including base cost, adjustments, fees, and contingency  
- Light and Dark mode for improved user experience  
- Real-time updates based on user input  

## How It Works

The system uses a rule-based calculation approach:

1. A base cost is calculated using building area and predefined rates  
2. Adjustment factors such as location and specification level are applied  
3. Additional costs (e.g. professional fees and contingency) are calculated as percentages  
4. A final total cost is generated and displayed to the user  

## How to Run Locally

1. Download or clone the repository  
2. Open the project folder in Visual Studio Code  
3. Open the main HTML file (e.g. `index.html`) in a browser
